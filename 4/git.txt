1) створюємо акаунт на одному із веб-сервісів (наприклад - https://github.com/)
2) cтворюємо респозиторій
3) встановлюємо git
4) виконуємо git clone 
5) переносимо наш сайт в створений проект
6) додаємо файли в респозиторій (git add .)
7) створюємо коміт (git commit -m "create project")
8) виконуємо заливку файлів в респозиторій (git push)
9) вводимо логін і пароль
10) переходимо в https://github.com
